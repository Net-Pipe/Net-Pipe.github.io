Aug 14, 2021

最大公因数 ¶

$a$ 和 $b$ 的最大公因数记作 $gcd(a,b)$，简记为 $(a,b)$.

设 $a > b$，$d$ 是 $a,b$ 的任意一个公因数，则：

\left\{\begin{aligned}
    a&=k_1d\\
    b&=k_2d
\end{aligned}\right.
\eq
a \bmod b=\Big\lfloor\frac{a}{b}\Big\rfloor\cdot b=\Big\lfloor\frac{k_1}{k_2}\Big\rfloor\cdot k_2d
\eq
d\mid(a \bmod b)

$∴a$ 和 $b$ 的所有公因数，同时也是 $b$ 和 $a \bmod b$ 的公因数，因此它们的最大公因数也相等.

$$gcd(a,b)=gcd(b,a \bmod b)$$

使用递归求解. 递归进行到 $b=0$ 时，$a$ 即为 $gcd(a,b)$.

时间复杂度为 $O(log{(a+b)})$.

int gcd(int a, int b) {
    if(b == 0) return a;
    return gcd(b, a % b);
}

最小公倍数 ¶

$a$ 和 $b$ 的最小公倍数记作 $lcm(a,b)$，简记为 $[a,b]$.

设 $gcd(a,b)=d$，则 $a=k_1d,b=k_2d$，所以：

$$lcm(a,b)=k_1k_2d=\frac{k_1d\cdot k_2d}{d}=\frac{ab}{gcd(a,b)}$$

时间复杂度为 $O(log{(a+b)})$.

int gcd(int a, int b) {
    if(b == 0) return a;
    return gcd(b, a % b);
}

int lcm(int a, int b) {
    return a * b / gcd(a, b);
}

扩展欧几里得算法 ¶

扩展欧几里得算法用于求方程 $ax+by=gcd(a,b)$ 关于 $x,y$ 的解，其中 $a,b$ 是常数.

设 $ax_1+by_1=gcd(a,b)$,

设 $bx_2+(a \bmod b)y_2=gcd(b,a \bmod b)$,

由 $gcd(a,b)=gcd(b,a \bmod b)$ 可知：

\begin{aligned}
ax_1+by_1&=bx_2+(a \bmod b)\,y_2\\
&=bx_2+(a-\lfloor\frac{a}{b}\rfloor\cdot b)\,y_2\\
&=bx_2+ay_2-\lfloor\frac{a}{b}\rfloor\cdot by_2\\
&=ay_2+b(x_2-\lfloor\frac{a}{b}\rfloor\cdot y_2)
\end{aligned}

$$∴\left\{\begin{aligned}x_1&=y_2\\y_1&=x_2-\lfloor\frac{a}{b}\rfloor\cdot y_2\end{aligned}\right.$$

自此，我们可以通过 $bx_2+(a \bmod b)\,y_2=gcd(a,b)$ 的解推出 $ax+by=gcd(b,a \bmod b)$ 的解. 原问题的求解就可以放进递归里.

那么这个递归的边界条件是什么呢？

当递归到 $b=0$ 时，$a=gcd(a,b)$，此时一定有一组解 $\left\{\begin{aligned}x&=1\\y&=0\end{aligned}\right.$ 满足 $ax+by=gcd(a,b)$. 这个特殊解可以作为递归的边界条件.

void exGcd(int a, int b, int &x, int &y) {
    if(b == 0) { x = 1, y = 0; return; }
    exGcd(b, a % b, x, y);
    int t = x; x = y, y = t - a / b * y;
}
